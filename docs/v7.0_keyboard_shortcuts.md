# Version 7.0 - Keyboard Shortcuts

## Übersicht

Version 7.0 führt umfassende Keyboard Shortcuts ein, um die Produktivität zu steigern und häufige Operationen zu beschleunigen.

## Verwendung

### Plot-Typ schnell wechseln

1. Drücken Sie **Ctrl+Shift+[1-7]** um direkt zum gewünschten Plot-Typ zu wechseln
2. Der Plot wird automatisch aktualisiert
3. Keine Maus erforderlich!

**Beispiel-Workflow:**
- Daten laden
- **Ctrl+Shift+1** → Log-Log Ansicht
- **Ctrl+Shift+2** → Porod Ansicht prüfen
- **Ctrl+Shift+3** → Kratky Ansicht prüfen
- Sehr schnelles Durchschalten möglich!

### Kurven schnell bearbeiten

1. Wählen Sie ein Dataset oder eine Gruppe im Tree
2. Drücken Sie **Ctrl+E**
3. Der entsprechende Editor öffnet sich:
   - Für Datasets: Kurven-Einstellungen-Dialog
   - Für Gruppen: Gruppen-Kurven-Editor

### Session-Management

- **Ctrl+S**: Schnelles Speichern der aktuellen Session
- **Ctrl+O** oder **Ctrl+Shift+S**: Session laden
- Workflow: Arbeiten, Ctrl+S drücken, fertig!

### Export

- **Ctrl+Shift+E**: Export-Dialog öffnen
- Schneller als über das Menü zu navigieren

## Implementierungs-Details

### Technische Umsetzung

Die Shortcuts sind in zwei Kategorien implementiert:

**1. Menü-Shortcuts (QAction.setShortcut)**
- Sichtbar im Menü neben dem Eintrag
- Beispiele: Ctrl+S, Ctrl+O, Ctrl+L, Ctrl+Shift+E

**2. Globale Shortcuts (QShortcut)**
- Funktionieren überall in der Anwendung
- Beispiele: Ctrl+Shift+1-7, Ctrl+E, Ctrl+G, Delete

### Konflikte vermeiden

Die Shortcuts wurden so gewählt, dass sie:
- Nicht mit Standard-System-Shortcuts kollidieren
- Intuitiv sind (z.B. E = Edit, L = Legend, G = Group)
- Mit Qt-Konventionen übereinstimmen

### Erweiterbarkeit

Das System ist einfach erweiterbar:

```python
def setup_shortcuts(self):
    # Neuer Shortcut hinzufügen
    new_shortcut = QShortcut(QKeySequence("Ctrl+N"), self)
    new_shortcut.activated.connect(self.neue_funktion)
```

## Tipps für Power-User

### Schneller Workflow

1. **Daten laden** (Maus)
2. **Ctrl+G** → Gruppe erstellen
3. **Ctrl+E** → Kurve anpassen
4. **Ctrl+Shift+1** bis **Ctrl+Shift+7** → Verschiedene Ansichten durchschalten
5. **Ctrl+L** → Legende anpassen
6. **Ctrl+Shift+E** → Exportieren
7. **Ctrl+S** → Session speichern

Alles ohne die Hände von der Tastatur zu nehmen!

### Tastatur-First-Ansatz

- Tree-Navigation: Pfeiltasten
- Auswahl: Enter
- Löschen: Delete
- Bearbeiten: Ctrl+E
- Plot wechseln: Ctrl+Shift+[1-7]
